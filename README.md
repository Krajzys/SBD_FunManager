# SBD_FunManager

# Opis funkcjonalności:
	Program umożliwiający zapisywanie filmów/seriali/gier/książek
	, ocenianie ich i umieszczanie na listach w zależności od stopnia
	ukończenia.

# Encje:
	1.Ukończone
	2.Oczekujące
	3.W Trakcie
	4.Użytkownicy
	5.Gry
	6.Książki
	7.Filmy
	8.Seriale
	9.Autorzy(książki)
	10.Producenci(gry)
	11.Reżyserzy(filmy/seriale)
